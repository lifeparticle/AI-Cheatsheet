1. Token — A full word, a fragment of a word, or nothing more than one character
2. Prompt — The input text given to a model to generate a response
3. System prompt — Instructions given to the model before the conversation begins
4. Context window — The maximum amount of text a model can process at once
5. Temperature — A setting that controls how random or creative the model's output is
6. Sycophancy — When a model agrees with the user rather than giving an honest answer
7. Embedding — A numerical representation of text used to measure meaning and similarity
8. Hallucination — When a model confidently produces false or made-up information
